## How to Use

1. Visit any of the websites above
2. Download your preferred ice cream PNG image
3. Save it to the `images/` folder in your project
4. Rename it to `hero-icecream.png`
5. The website will automatically use it!

## Current Setup

Your `index.php` is configured to use:
- Primary: `images/hero-icecream.png` (you need to download and save this)
- Fallback: Online PNG from freeiconspng.com (loads automatically if local file not found)
